# EduAssist Design System

## Design Philosophy

EduAssist should feel like a premium iOS application.

Inspired by:

* Apple Wallet
* Apple Fitness
* Things 3
* Linear
* Craft

Avoid:

* Generic ERP dashboards
* Android Material templates
* Heavy glassmorphism
* Excessive gradients

## Color System

Brand Accent:

* Teal (#14B8A6)
* Cyan (#06B6D4)

Functional Colors:

* Attendance = Green
* Homework = Violet
* Fees = Amber
* Tests = Blue
* Notices = Orange

## Typography

Display XL = 36
Display L = 30
Heading = 24
Title = 18
Body = 16
Caption = 13
Meta = 11

## Elevation

Level 0 = Background
Level 1 = Content Cards
Level 2 = Hero Cards
Level 3 = Floating Elements

## Navigation

Parent App:

* Consumption focused

Teacher App:

* Action focused

Admin Portal:

* Desktop-first
* Table-centric
